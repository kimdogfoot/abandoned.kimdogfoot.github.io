리눅스에서 자주 사용하는 네트워크 커맨드

리눅스에서 네트워크 상태를 확인하기 위해 자주 사용하는 커맨드

Listening port 찾기

ss -tlp
-t : TCP 커넥션 보기
-l : Listen 중인 것만 보기
-p : 프로세스정보까지 보기
아래 예제는 4000 포트로 hexo 를 돌리고 있는 내용이 출력된 결과이다.

pid=22388 인 놈이 4000 port를 사용하고 있는 것을 알 수 있다.
※ 이 블로그를 만드는데 사용되는 hexo라는 블로그 프레임워크 이다.
